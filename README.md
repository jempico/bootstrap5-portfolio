## Installation

To get a local copy up and running follow these simple steps.

1. Clone the repo
   ```sh
   git clone https://github.com/jempico/bootstrap5-portfolio.git
   ```
2. Install NPM packages
   ```sh
   npm install
3. For watching and compiling SASS type:
   ```sh
    npm run compile:sass
   ```
